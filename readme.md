# PixiJS Playables Collection

Welcome to the collection of interactive playables built with **PixiJS**!
This collection of interactive playables, developed with PixiJS and designed to meet ad platform limitations, is optimized for lightweight (<5MB).

---

## 🔹 Features

- Minimal file and asset sizes, optimized for playables.
- Adaptive for mobile devices and Retina display.
- Compressed assets and minified scripts.
- Interactive tooltips and animations (swipes, taps, effects).
- Support for smooth animations powered by GSAP.
